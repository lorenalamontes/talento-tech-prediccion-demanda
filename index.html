Paso 1 — Datos

Pipeline ETL

Extracción, transformación y carga del historial de ventas de Mundipartes (jun 2025 – may 2026): 12 meses, múltiples asesores, marcas y artículos.

🔵

EXTRACT

  • Archivo Excel de ventas históricas
  • Hoja Cliente ajustado
  • Columnas: Asesor, Cliente, Marca, Artículo + 12 meses
  • Fuente: CRM / ERP / hojas de asesores
🟡

TRANSFORM

  • Nulos → imputados con 0
  • Negativos (devoluciones) → clip(lower=0)
  • Variables derivadas: total, meses activo, promedio mensual
  • Normalización con StandardScaler para K-Means
🟢

LOAD

  • DataFrame limpio listo para EDA
  • Tabla de clientes agrupados
  • Features estandarizadas para ML
  • Input al agente de recomendación

# Normalizar features para K-Means X_cluster = clientes[['total_compras', 'meses_activo', 'num_articulos']] scaler = StandardScaler() X_scaled = scaler.fit_transform(X_cluster) kmeans = KMeans(n_clusters=3, random_state=42, n_init=10) clientes['segmento'] = kmeans.fit_predict(X_scaled) # Renombrar segmentos según volumen promedio medias = clientes.groupby('segmento')['total_compras'].mean().sort_values() mapa = {medias.index[0]: 'Bajo Potencial', medias.index[1]: 'Potencial Medio', medias.index[2]: 'Alto Potencial'}

Resultados

Conclusiones del Proyecto

Lo que aprendemos de los datos y cómo transforma la operación comercial de Mundipartes.

Conclusión 01 · Segmentación

El modelo K-Means identificó exitosamente 3 segmentos de clientes con perfiles de consumo claramente diferenciados: Alto, Medio y Bajo Potencial. Esto permite priorizar el tiempo del asesor en los clientes con mayor probabilidad de compra inmediata.

Conclusión 02 · Predicción

La Regresión Lineal confirmó que el historial de los 11 meses previos es un predictor significativo del consumo del mes siguiente. El modelo es técnicamente viable para integrarse al ciclo comercial real de Mundipartes.

Conclusión 03 · Distribución

El análisis EDA reveló una distribución tipo Pareto: un grupo reducido de clientes genera la mayor parte del volumen de ventas. Identificar y proteger este núcleo es estratégicamente prioritario.

Conclusión 04 · Impacto comercial

Al combinar ambos modelos, el asesor puede llegar al cliente antes de que este necesite el producto, aumentando la tasa de conversión, reduciendo quiebres de inventario y fortaleciendo la fidelización de la marca.
